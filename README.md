## 🌷 피부묘기증 분석 및 관리 CDSS

> 데이터 기반으로 만성 가려움증(피부묘기증)을 분석하고 질환을 객관적으로 판단하는 시스템

---

### 📌 Overview
피부질환은 개인차가 크고 주관적 판단에 의존하는 경우가 많습니다.  
본 프로젝트는 이미지 분석과 증상 데이터를 결합하여  
질환을 객관적으로 판단하고 관리 가능성을 탐색합니다.

---

### 🎯 Key Features
- 🧠 이미지 기반 질환 분석 (VGG16, Cosine Similarity)  
- 🌿 증상 기반 분류 모델 (Decision Tree)  
- 💊 치료 패턴 분석 (환자 데이터 기반)  
- 📱 증상 기록 및 이미지 분석 앱 설계  

---

### 🛠 Tech Stack
Python / Pandas / NumPy / Scikit-learn / VGG16

---

### 📊 Results
- 피부묘기증과 가장 높은 유사도 (0.71)  
- 팽진(Wheal)이 핵심 분류 변수  
- 중증 환자일수록 약물 사용 증가  

---

### 💡 Insight
데이터 기반 접근을 통해 주관적인 피부 증상을 객관화할 수 있으며,  
향후 개인 맞춤형 관리 시스템으로 확장 가능성을 확인했습니다.
